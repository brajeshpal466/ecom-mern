# Frontend Setup & Run

## 1. Install Dependencies

```bash
npm install
```

---

## 2. Start the Application

```bash
npm start
```

---

## 3. Open in Browser

```text
http://localhost:3000
```

---

## ⚠️ Important

* Make sure backend is running on:

```text
http://localhost:5001
```

---

## (Optional) Environment Setup

Create `.env` file if needed:

```env
REACT_APP_API_BASE_URL=http://localhost:5001/api
```
